## 插件获取方式：

1. 官网下载

[应用市场](https://halo.run/store/apps?type=PLUGIN)

2. Github 下载

[GitHub - halo-sigs/awesome-halo: 与 Halo 相关的周边生态资源列表](https://github.com/halo-sigs/awesome-halo#%E6%8F%92%E4%BB%B6)

[插件](https://bbs.halo.run/t/plugins)
